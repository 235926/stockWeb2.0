# 股权管理系统
- 以 Vue2.O 为主要版本
- npm i 安装依赖
- npm run dev 启动项目
- npm run build 打包项目

### 系统配置
- 在 src/pinia/systemConfig.js 中修改系统配置，如（布局方案，主题色等）
